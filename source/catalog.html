Two directories will be created in outdir: outputs, which will contain the output catalog and logfile, and plots, which will contain plots of the redshift distribution, color-color diagram, and selection window diagnostics.

Parameters:
  • nsrc (int) – Number of synthetic sources to create
  • add_lines (bool) – If True, emission lines will be added to the spectra of the sources.
  • outdir (Optional[str]) – Directory for simulation outputs (catalogs and plots). Defaults to current working directory.
build_catalog()¶

Builds the catalog of synthetic photometry.

All physical parameters are assigned by the Source class. Filter.calc_synphot() converts the synthetic spectrum into flux densities in each filter. These in turn are converted into AB magnitudes. The output is output in both FITS and ascii formats in the outputs directory.

generate_redshifts(nsrc, lowz=0.1, highz=8.5, plotdist=True)¶

Creates a flat redshift distribution of sources in range lowz < z < highz.

Redshifts are pulled from a uniform distribution using numpy.random.uniform().

Parameters:
  • nsrc (int) – Number of sources
  • lowz (Optional[float]) – Lower redshift of distribution. Defaults to 0.1
  • highz (Optional[float]) – Upper redshift of distribution. Defaults to 8.5
  • plotdist (Optional[bool]) – If true, plots the generated distribution as a histogram and saves to plots/zdist.pdf.

zz – Array of randomly generated redshifts.

get_luminosity_functions()¶

Creates instances of the LFs appropriate for each redshift range.

The luminosity of each source is pulled randomly from the appropriate luminosity function. In each redshift range, the cumulative LF is calculated by integrating the LF from \(10L^*\) down to the luminosity corresponding to the \(1\sigma\) magnitude limit of our survey. The magnitude limits are the median \(1\sigma\) limits in the filters covering the LFs’ characteristic wavelengths in the redshift ranges.
