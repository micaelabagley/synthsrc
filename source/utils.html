utils module¶

class synthsrc.utils.Cosmology(h=0.678, omega_m=0.308, omega_l=0.692)¶

Bases: object

A class to set the cosmology and calculate useful quantities.

Input parameters (Hubble constant, matter and dark energy density parameters) are based on the Planck 2015 results

Parameters:
  • h (Optional[float]) – Hubble constant in units of 100 km/s/Mpc. Defaults to 0.678
  • omega_m (Optional[float]) – Matter density parameter. Defaults to 0.308
  • omega_l (Optional[float]) – Dark energy density parameter. Defaults to 0.692
get_lookback_time(z)¶

Calculates the lookback time to redshift z.

Uses cosmolopy.distance.age() and cosmolopy.distance.lookback_time(), which are based on equation 30 of David Hogg’s Distance measures in cosmology.

Parameters:z (float) – Redshift for which to calculate the lookback time
Returns:Lookback time in years
Return type:(float)
